🔗 BACKEND API ENDPOINTS:

Discord:
  GET /api/discord               - Tüm veriler
  GET /api/discord/:id           - Tek kayıt

Mail:
  GET /api/mail                  - Tüm veriler
  GET /api/mail/search?q=query   - Arama

Facebook:
  GET /api/facebook              - Tüm veriler
  GET /api/facebook/stats        - İstatistikler

Valorant:
  GET /api/valorant              - Tüm veriler
  GET /api/valorant/search?q=q   - Arama

Instagram:
  GET /api/instagram             - Tüm veriler
  GET /api/instagram/count       - Sayı

Epic Games:
  GET /api/epicgames             - Tüm veriler
  GET /api/epicgames/search?email=q - E-mail arama

CraftRise:
  GET /api/crafrise              - Veriler

API Info:
  GET /api/info                  - Endpoints hakkında
