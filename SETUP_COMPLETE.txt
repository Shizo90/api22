📊 SUNULAN VERİ KAYNAKLARI:
═══════════════════════════════════════════════════════════════════

1. 💬 DISCORD
   Endpoint: /api/discord
   Format: SQL
   Operasyonlar: List, Get by ID

2. 📧 MAIL
   Endpoint: /api/mail
   Format: JSON
   Operasyonlar: List, Search

3. 👥 FACEBOOK
   Endpoint: /api/facebook
   Format: CSV
   Operasyonlar: List, Stats

4. 🎮 VALORANT
   Endpoint: /api/valorant
   Format: JSON
   Operasyonlar: List, Search

5. 📸 INSTAGRAM
   Endpoint: /api/instagram
   Format: JSON
   Operasyonlar: List, Count

6. 🎮 EPIC GAMES
   Endpoint: /api/epicgames
   Format: TXT (Email:Password)
   Operasyonlar: List, Search by Email

7. ⛏️ CRAFRISE
   Endpoint: /api/crafrise
   Format: Text
   Operasyonlar: List

🚀 BAŞLANGIÇ:
═══════════════════════════════════════════════════════════════════

1. Bağımlılıkları Kur:
   $ npm install

2. Sunucuyu Başlat:
   $ npm start

3. Test Et:
   $ node test_api.js
   
   veya tarayıcıda açın:
   http://localhost:3000/

📚 BAŞLICA ENDPOINTS:
═══════════════════════════════════════════════════════════════════

Root:
  GET /                          - API hakkında bilgi

Discord:
  GET /api/discord               - Tüm veriler
  GET /api/discord/:id           - Tek kayıt

Mail:
  GET /api/mail                  - Tüm veriler
  GET /api/mail/search?q=query   - Arama

Facebook:
  GET /api/facebook              - Tüm veriler
  GET /api/facebook/stats        - İstatistikler

Valorant:
  GET /api/valorant              - Tüm veriler
  GET /api/valorant/search?q=q   - Arama

Instagram:
  GET /api/instagram             - Tüm veriler
  GET /api/instagram/count       - Sayı

Epic Games:
  GET /api/epicgames             - Tüm veriler
  GET /api/epicgames/search?email=q - E-mail arama

CraftRise:
  GET /api/crafrise              - Veriler

🛠️ TEKNOLOJI STACK:
═══════════════════════════════════════════════════════════════════

Server:
  • Node.js >= 12.0.0
  • Express.js 4.18.2
  • CORS middleware
  • Body Parser

Veri Formatları:
  • JSON
  • CSV
  • SQL
  • Plain Text
